Coherence

A client/server mod synchronizer for Minecraft

View on GitHub 1.8a01 (recommended) View all downloads

Modpacks made optional

Simple

Coherence remove the hassle of ensuring you have the necessary mods to connect to a modded server. All you need is one mod to handle getting all the other mods

As for server owners, you only need to do one thing to support Coherence: drag the mod into your mods folder. That's it. It'll handle all the distribution for you.

Efficient

Coherence caches the mods on the client side, so when the server updates its mods, Coherence only downloads the new mods.

Dynamic

Coherence automatically pulls the config files from the server, too. No manual mod configuring required.
